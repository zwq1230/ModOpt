Check input object is callable

This method checks if the input operator is a callable funciton and optionally adds support for arguments and keyword arguments if not already provided

  • add_agrs (bool, optional) – Option to add support for agrs and kwargs (default is True)

modopt.base.types.check_float(val)[source]¶

Check if input value is a float or a numpy.ndarray of floats, if not convert.

Parameters

val (any) – Input value

Returns

Input value as a float

Return type

float or numpy.ndarray

Examples

>>> from modopt.base.types import check_float
>>> a = np.arange(5)
>>> a
array([0, 1, 2, 3, 4])
>>> check_float(a)
array([ 0.,  1.,  2.,  3.,  4.])
