1.1.6.1.1. modopt.signal.filter module¶

FILTER ROUTINES

This module contains methods for distance measurements in cosmology.

Author

Samuel Farrens <samuel.farrens@cea.fr>

modopt.signal.filter.Gaussian_filter(x, sigma, norm=True)[source]¶

Gaussian filter

This method implements a Gaussian filter.

Parameters

  • x (float) – Input data point

  • sigma (float) – Standard deviation (filter scale)

  • norm (bool) – Option to return normalised data (default is True)

Returns

Gaussian filtered data point

Return type

float

Examples

>>> from modopt.signal.filter import Gaussian_filter
>>> Gaussian_filter(1, 1)
0.24197072451914337

>>> Gaussian_filter(1, 1, False)
0.60653065971263342
